```python
class Solution(object):
    def groupAnagrams(self, strs):
        hash=defaultdict(list)
        for word in strs:
            count=[0]*26
            for ch in word:
                count[ord(ch)-ord('a')]+=1
            hash[tuple(count)].append(word)
            return list(hash.values())
            
```
‘str’.join()不是+join能把单个字符组合成字符串
sorted（）可以应用于数字列表，字符串，字典
字符串返回的是单个字符组成的列表，直接字典返回的是键的排序
defaultdict（list）提供空列表组成的字典，若赋值时没有这个键会自动加上，正常字典会错误
hash.values（）返回的是值的动态字典试图形如dict_values([['eat', 'tea'], ['tan']])要用list()化为列表
字典不能用列表作为键，所以要用tuple（）
count=[0]\*26
26个字母的计数数组，用来记录对应单词的出现次数
